How It Works

Built for developers,
loved by teams

From a single agent to a fleet of thousands, Splinter scales with you.

Define Your Agents in Code

Write agent behaviors using our intuitive SDK. Support for Python, TypeScript, Go, and Rust.

import { Agent, Swarm } from '@splinter/sdk'; const researcher = new Agent({ name: 'ResearchAgent', model: 'gpt-4-turbo', tools: ['web_search', 'analyze'], memory: true, }); const swarm = new Swarm({ agents: [researcher], instances: 100, coordination: 'distributed', }); await swarm.deploy();
